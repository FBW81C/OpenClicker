# OpenClicker

**OpenClicker** is a free and open-source auto clicker and key presser for Windows.  

## Overview

OpenClicker allows automated execution of mouse clicks and key presses with fully configurable intervals, repetitions, delays, and action patterns.  
All functionality runs **locally, offline, and without telemetry**.

## Features

### Timing Control
- Action interval (hours, minutes, seconds, milliseconds)
- Action repeat (infinite or fixed amount)
- Start delay (hours, minutes, seconds, milliseconds)
- Duration (only for click type = hold)

### Mouse Actions
- Mouse button: left, right, middle
- Click type: single, double, hold
- Click position:
  - current position
  - fixed X/Y coordinates
  - location picker

### Keyboard Actions
- Any key
- Key down / key up

### Action Patterns
- Single action (mouse only)
- Multiple actions in sequence (pattern mode)

### Profiles
- Export profiles
- Import profiles
- Set profiles as default

### Hotkeys
- Start
- Stop
- Toggle
- Configurable hotkey bindings

### Action Recorder
- Record mouse and keyboard input
- Replay recorded sequences

## Dependencies
- **.NET 8 (Core)**

## Platform
- Windows
