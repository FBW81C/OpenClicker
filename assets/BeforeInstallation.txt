Welcome to the OpenClicker Installer!

OpenClicker is a simple free and open-source Open Clicker and Key Presser for Windows by FBW81C.

GitHub: https://github.com/FBW81C/OpenClicker
YouTube: https://www.youtube.com/@FBW81C

Features:
- Action Interval (Hours, Minutes, Seconds, Milliseconds)
- Action Repeat (Infinite, Amount)
- Click Options
    - Mouse Button (Left, Right, Middle)
    - Click Type (Single, Double, Hold)
    - Clicking Position (Current, XY)
- Key Options
    - Key
    - Up / Down
- Starting Delay (Hours, Minutes, Seconds, Milliseconds)
- Duration (Only for Click Type = Holding)
- Pattern
    - Single (only Mouse)
    - Multiple (Multiple actions in a row)
- Hotkeys
    - Start, Stop, Toggle
    - Hotkey Settings
- Action Recorder

Dependencies:
- .NET 8 (Core)
